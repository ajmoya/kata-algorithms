# kata-algorithms

Dado un <b>string</b> formado por <b>(), [] y {}</b>, escribe un programa que indique si los <b>pares correspondientes son correctos.</b>


## Output examples
Input: [()]{}{[()()]} => CORRECTO

Input: [(]{) => INCORRECTO
